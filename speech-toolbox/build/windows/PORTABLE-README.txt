语音工具箱 — Windows 免安装版
==============================

功能：语音转文字、文字转语音

1. 解压本 zip 到任意目录（如桌面、D:\Tools、U 盘）
2. 进入 SpeechToolbox 文件夹
3. 双击 SpeechToolbox.exe 即可使用

注意：
- 请勿单独移动 SpeechToolbox.exe，需与同目录下其他文件放在一起
- 首次运行若提示缺少运行库，请安装 VC++ 2015-2022 x64：
  https://aka.ms/vs/17/release/vc_redist.x64.exe
- 语音识别首次使用会自动下载 Whisper 模型，请保持网络畅通
- 处理视频文件需单独安装 ffmpeg 并加入 PATH
- 文字转语音需要联网（使用 Microsoft Edge 语音服务）
- 删除整个 SpeechToolbox 文件夹即可卸载，不会在系统中留下安装记录
