PDF 工具箱 — Windows 免安装版
==============================

功能：PDF 转 Word、拼接 PDF、拆分 PDF

1. 解压本 zip 到任意目录（如桌面、D:\Tools、U 盘）
2. 进入 PdfToWord 文件夹
3. 双击 PdfToWord.exe 即可使用

注意：
- 请勿单独移动 PdfToWord.exe，需与同目录下其他文件放在一起
- 首次运行若提示缺少运行库，请安装 VC++ 2015-2022 x64：
  https://aka.ms/vs/17/release/vc_redist.x64.exe
- 删除整个 PdfToWord 文件夹即可卸载，不会在系统中留下安装记录
