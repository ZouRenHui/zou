录屏 / 截屏工具 — Windows 免安装版
==================================

功能：屏幕录制、全屏截屏、区域截屏、复制到剪贴板

1. 解压本 zip 到任意目录（如桌面、D:\Tools、U 盘）
2. 进入 ScreenCapture 文件夹
3. 双击 ScreenCapture.exe 即可使用

注意：
- 请勿单独移动 ScreenCapture.exe，需与同目录下其他文件放在一起
- 首次运行若提示缺少运行库，请安装 VC++ 2015-2022 x64：
  https://aka.ms/vs/17/release/vc_redist.x64.exe
- 录制系统声音需安装 ffmpeg，并在 Windows 声音设置中启用「立体声混音」
- 删除整个 ScreenCapture 文件夹即可卸载，不会在系统中留下安装记录
